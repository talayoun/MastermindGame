# Mastermind Game Engine
A robust desktop implementation of the classic Mastermind board game.

### Technical Highlights
* **Architecture:** Separation of concerns between `GameLogicManager` and WinForms UI.
* **Event-Driven:** Uses C# Events and Delegates for decoupled communication.
* **Dynamic UI:** Responsive game board based on user settings.

### Technologies
* **Language:** C#
* **Framework:** .NET WinForms
